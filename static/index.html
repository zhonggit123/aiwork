核心功能亮点

智能 Word/PDF 全解析

结合大模型多模态能力，精准提取单选、填空、阅读理解等题型。不仅支持 Word 和原生 PDF，更完美支持印刷扫描版 PDF、内嵌图片及复杂表格转高清大图等难点场景。

TTS 听力语音合成

内置豆包、有道、微软 Edge 多种 TTS 服务。可自动过滤音标，支持双人对话识别（男声/女声交替），一键合成标准听力音频。

无缝网页 DOM 填充

专为各大录题系统（如驰声等）优化的浏览器扩展。自动识别页面结构、富文本编辑器（UEditor等）及上传框，实现真正意义的“一键填入”。

技术路线与架构

1

本地解析服务 (Python/FastAPI)

基于 Python 的底层解析引擎：

  • 文档精细处理：使用 python-docx 和 PyMuPDF/pdf2image 提取段落文本，自动处理跨行/列的复杂表格并将其高清渲染为图片，同时提取内嵌的所有图片资源。
  • 大模型结构化抽取：通过自定义的 Prompt 对接各种主流大语言模型（OpenAI 格式支持），结合多模态视觉能力（VLM），将杂乱文本甚至是印刷扫描版 PDF转化为标准化的试题 JSON 数组。
  • 听力语音全自动生成：内置豆包、有道、微软等 TTS 引擎的代理接口，可智能剥离文本中的音标字符，精准处理中英混合及多角色对话场景。
2

浏览器扩展 (Manifest V3)

连接本地解析端和目标录入系统的桥梁：

  • 动态表单嗅探：Content Script 注入到各大在线题库平台，通过启发式正则与 DOM 分析，精准锁定"题干"、"选项"、"解析"等编辑区域（兼容 UEditor 等各种富文本编辑器）。
  • 现代化控制台：基于 Tailwind CSS 构建的扩展 Popup 界面，便于快捷配置 API 秘钥及各项生成参数（如自定义发音人及音量等）。
3

多模态协同自动填充

实现“所见即所得”的无感录入体验：

  • 附件劫持上传：自动将本地处理好的图片与表格转换回前端的 File 对象，模拟真实用户操作，精准派发给目标网站的图片/音频上传框。
  • 复杂题型自适应：针对带子题的复合题型（如阅读理解、完形填空、听力篇章），按层级自动切分并注入对应的题目空位及音频节点，释放双手。

使用教程

1

启动后端服务

双击运行打包好的 题库后端.exe（或通过 python app.py 启动）。运行后会自动开启 8766 端口，此时你正看到的这个网页即为本地服务。

2

安装浏览器扩展

打开 Chrome/Edge 浏览器，进入扩展管理页 (chrome://extensions/)，开启"开发者模式"，选择"加载已解压的扩展程序"，选中解压后的 extension 文件夹。

3

配置模型与选项

点击浏览器右上角的扩展图标，打开设置面板。填写好你的 LLM API Key（支持 OpenAI 格式，如火山引擎、DeepSeek 等），选择需要的 TTS 供应商及音色配置。

4

一键解析填充

在浏览器的录题系统页面直接点击扩展面板的「开始检测」，系统会读取表单结构。拖入 Word / PDF 文档后，AI 会自动解析并一键帮你填好题干、选项、答案及合成音频！
