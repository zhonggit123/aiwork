# 题库自动填表 - 插件安装与显示

## 1. 安装插件（加载已解压的扩展程序）

1. 打开 Chrome，在地址栏输入并回车：
   ```
   chrome://extensions/
   ```
2. 打开右上角 **「开发者模式」**。
3. 点击 **「加载已解压的扩展程序」**。
4. 在弹窗里选择**本项目的 `extension` 文件夹**（即包含 `manifest.json` 的文件夹）：
   - 路径示例：`/Users/zhong/Downloads/AI录题/extension`
5. 确认后，列表里会出现 **「题库自动填表」**，说明安装成功。

## 2. 在浏览器里看到插件图标

安装后图标可能被收起到「扩展程序」菜单里，需要**固定到工具栏**：

1. 点击 Chrome 右上角 **拼图图标**（扩展程序入口）。
2. 在列表里找到 **「题库自动填表」**。
3. 点击其右侧的 **图钉图标**，把它固定到工具栏。
4. 固定后，工具栏会显示插件图标，在录题页面（如 cp.91tszx.com）点击即可使用。

## 3. Word 多题与录题页「一题一题」如何兼容

录题页面是**一题一题**录入，Word 里是**多题**，插件会这样兼容：

- 上传 Word 后，后端会解析出**多道题**（题目列表）。
- 插件会**按顺序**：填当前页这一题 → 点「保存」→ 点「下一题」→ 等页面切到下一题 → 再填下一题……循环直到全部填完。
- 因此无需手动一题题切，只要在**选项**里配置好「保存/提交按钮」和「下一题按钮」（若页面是「保存」+「下一题」两个按钮，请都填上），即可一次上传 Word、自动逐题填入。

## 4. 使用前确认

- 本地服务已启动：在项目目录执行  
  ` .venv/bin/uvicorn app:app --port 8766`
- 录题页 URL 特征：若你们的录题地址不是默认的 `chisheng`，请在插件 **选项** 里设置「录题页 URL 特征」（如 `91tszx` 或 `cp.91tszx`），这样在你们页面打开时才会显示「已检测到录题页面」。
